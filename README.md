# Travel Tracker

Travel Tracker is a full-stack application that allows users to explore countries, mark visited destinations and manage favorite locations.

## Planned Features

* Browse countries from a public API
* Mark countries as visited
* Add countries to favorites
* Search and filter countries
* View personal travel statistics

## Technology Stack

### Backend

* Java 17
* Spring Boot
* Maven

### Frontend

* Angular 18
* TypeScript

### Database

* PostgreSQL
